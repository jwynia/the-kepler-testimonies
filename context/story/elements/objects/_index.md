Significant items in The Kepler Testimonies.

## Roster

*No objects created yet.*

Objects will be developed as story needs emerge.

## Usage

Only create object files for items that:
- Appear in multiple testimonies
- Have symbolic or plot significance
- Need consistent description across the story

Don't create files for ordinary items that don't need tracking.

## Object File Template

When creating an object file:

```markdown
# Object Name

Brief description.

## Physical Description
What does it look like? How big? What material?

## Significance
Why does this matter to the story?

## Key Details
- History
- How characters interact with it

## Relationship Network
- **Owned by**: [[character references]]
- **Located at**: [[place references]]
- **Appears in**: [[testimony references]]
```
